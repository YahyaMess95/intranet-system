# Integration 
## Tools :
 * Apache
 * Mysql
 * Php 5th version +
## How to integrate
After installing the tools
1- Import the database located in sql folder 
2- Locate the folder of the project in the apache front folder (htdocs in xampp / www in wampp) 
3- Run apache
